# text-search-engine-cpp

A console-based mini search engine built in C++ that processes a text file and allows users to search for word frequencies.

## Features
- Reads text from a file
- Converts text to lowercase and removes punctuation
- Counts word frequencies using hash maps
- Displays all word frequencies
- Shows top 5 most frequent words
- Menu-driven interface

## Technologies Used
- C++
- STL (unordered_map, vector, algorithm)
- File handling
- Lambda functions
